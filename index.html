Why This Platform Exists

Healthcare quality improves when patients are heard. This platform allows citizens to evaluate hospitals based on real experiences, encouraging accountability, transparency, and continuous improvement.

To ensure trust and prevent fake reviews, only verified users are allowed to submit evaluations.

User Login

Login is required before submitting any hospital evaluation. This helps maintain the credibility of ratings.

Name must contain letters only (no numbers).
Minimum 8 characters, uppercase, lowercase, number, and symbol.

How Hospital Evaluation Works

  • Users login using a validated identity
  • Hospitals are evaluated using multiple criteria
  • Ratings are calculated dynamically
  • Analytics summarize public feedback
  • No page reloads — everything updates instantly

Security & Data Integrity

This system applies strict validation rules to ensure that submitted data is logical, meaningful, and trustworthy.

  • Names with numbers are rejected
  • Weak passwords are blocked
  • Unauthenticated users cannot evaluate hospitals
  • User actions are controlled via JavaScript logic
